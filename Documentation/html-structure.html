HTML Structure

Structure of the HTML pages.

HTML Files

Here is the general structure of the HTML pages (.html files such as index.html):

  • The HTML head Tag contains meta data for SEO, CSS include and social networks meta data.
  • The navigation bar (at top) of a page is nested within a tag with class navbar-top. It includes the logo (left), the menu icon, and the menu links .
  • The sidebar navigation (at left), located within the nav-sidebar.
  • The background cover of the page is nested within a div tag with a class cover-page.
  • The main content of the page is nested within a tag with a class page-main. It contains one or several section, represented by tag (a section or a div) with class section.
  • The footer of the page, containing basic information such as copyright notice or social networks links. It is located within the footer tag with class footer-page.
  • And the scripts files required by the page such as Bootstrap.

Head tag - top

The <head> tag contains the Title and description of the tag, used by search engine to identify your website. It may also include, third-party meta, such as social network meta (mainly Facebook and Twitter) helping them to describe your website.

Change the title and the description of your page by editing the title tag value, and the content value of the tag <meta name="description" content="YOUR DESCRIPTION">. Additionaly, you can add Facebook Opengraph data or Twitter metadata for better compatibility with these social networks.

Required CSS files (such as main stylesheet, web fonts, bootstrap, ...) are also included within the <head> tag. Add them by using the link tag such as :
<link rel="stylesheet" href="...url-to-css-file.css">.

Parameters

A page-main may have the following parameters :

  • Scroll navigation
  • Scroll animation
  • Slideshow scroll with fullpage.js
  • Slideshow scroll with CSS
  • Locomotive scroll

Scroll navigation

Class sections-scroll will add dots navigation at right of the page-cover when a nav with class nav-dot-menu is added after the navigation bar elements.

It will also add class active to an element with an attribute data-section-anchor="SECTION_ID" when the section with id="SECTION_ID" is visible on screen.

Scroll animation

Class main-anim will enable animation on page scroll.

Slideshow scroll with fullpage js

Optional class page-fullpage can be added as class of the <main> tag to get slideshow effect behavehior on page scroll by using fullpage.js v2 plugin.

This requires that all the sections of the page should have unique and different data-section attribute values and should not have an id attribute. You may also need to add class fp-auto-height-responsive to each section to get a better render result on mobile devices.

Page footer - top

At bottom of the page, the div with class footer-page embeds the social links (ul with class social), and additional website footer notice (such as copyright).

Social links icons here use ionicons and font awesome icons.

Fontawesome icons are presented by a <a href="..."><i class="fa fa-SOCIALNET"></i></a> line.SOCIALNET value can be "facebook", "twitter", ...

For Ionicons, they are presented by a <a href="..."><i class="ion ion-logo-SOCIALNET"></i></a> line.SOCIALNET value can be "facebook", "twitter", ...

Duplicate these line and edit SOCIALNET by a custom one (pinterest, tumblr, ...) if you add more than the defaults links in the template.

Legal notice are located withini the div with class footer-left.
