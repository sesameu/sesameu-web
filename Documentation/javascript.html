Javascript files

Javascript makes this theme more dynamic and animated, even thought some animations use CSS3.

This theme imports these great vendor specific Javascript Plugin (Thanks to their authors) :

  • Bootstrap : Bootstrap 5 Javascript
  • Swiper : A cool plugin to make image slider
  • Rellax : A plugin to make parallax animation. It uses a modified version of Rellax.
  • jQuery : Used with the fullpage.js plugin.
  • scrolloverflow.js" : Used with the fullpage.js plugin.
  • fullpage.js Plugin : A jQuery plugin to make page vertical carousel slider

It also uses our home-made JS plugins

  • scroll-slider.js : An horizontal slider which react on vertical scrolling
  • section-scroll.js : A plugin that add dot navs, and watch page scrolling to modify section states (active, animation, content color, ...).
  • form_script.js : to handle Ajax Forms

Based on these plugins, it uses the following javascripts main files :

  • main.js : contains the general script of the theme (plugin initialization, smooth scroll, page loader)
  • fullpage-init.js : init fullpage.js carousel slider plugin

Their are ready to use and it is not suggest to modify them but you can always tweak them (such as for a responsiveness case or to replace the backend AJAX server).
